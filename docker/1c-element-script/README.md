# 1C:Enterprise.Element Script

[**1С:Предприятие.Элемент Скрипт**](https://lang.1c.ru/) - Среда выполнения для скриптов, написанных на языке 1С:Элемент. Подходит для быстрой разработки и обучения. Оснащен средствами DevOps и CLI.

## Сборка образа

Все переменные перечислены в файле [.arg.tmpl](build/.arg.tmpl)

Значимые переменные среды:

- `OC_EXECUTOR_VERSION` - версия 1С:Элемент.Скрипт, которую нужно установить.
