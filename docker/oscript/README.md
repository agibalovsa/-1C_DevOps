# OneScript

[**OneScript**](https://oscript.io/) - Независимая кроссплатформенная реализация виртуальной машины, исполняющей скрипты на языке 1С:Предприятие 8.

## Сборка образа

Все переменные перечислены в файле [.arg.tmpl](build/.arg.tmpl)

Значимые переменные среды:

- `OSCRIPT_VERSION` - версия OneScript, которую нужно установить.
